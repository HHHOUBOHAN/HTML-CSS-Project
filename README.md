 # 小米官网首页 - 静态仿制

小米官网 (mi.com) 首页的 HTML + CSS + Swiper 复刻练习项目。  
仅用于学习和展示，**非商业用途**。

## 技术

- HTML5 + CSS3  
- 轮播图  
- JavaScript  
- Flex 布局 + 媒体查询  
- 滚动后导航变色

## 功能

- 透明 → 黑色导航栏（滚动后变色）  
- 多组全屏轮播图  
- 响应式（手机端隐藏菜单）  
- 小米风格橙色 hover 效果  
- 长 footer（含备案信息）

## 如何运行

1. 下载或克隆仓库  
2. 直接用浏览器打开 `index.html`  
   或  
3. 用 VSCode Live Server / `python -m http.server`
